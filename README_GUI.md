# 注意事项
请先在cookie.txt文件内填入黑胶会员账号的cookie 才可以解析！
Cookie格式为↓
MUSIC_U=你获取到的MUSIC_U值;appver=8.9.70; 完整填入cookie.txt即可！
具体值在cookie.txt里面就有 替换一下就行了
